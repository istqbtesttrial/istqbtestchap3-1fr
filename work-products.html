Revue Interactive : Identifiez les Problèmes

Examinez la description suivante d'un produit d'activité. Mettez en évidence les parties qui, selon vous, pourraient être améliorées ou poser problème :

  • La base de données doit pouvoir stocker un nombre illimité d'enregistrements utilisateur.
  • Les mots de passe seront chiffrés à l'aide d'un algorithme réversible simple pour faciliter leur récupération.
  • Le système doit pouvoir gérer jusqu'à 100 utilisateurs se connectant simultanément.
